# 🧠 AI Quizer

AI Quizer is a web application that allows users to generate custom quizzes from videos or documents. Designed for students, educators, and lifelong learners, the app uses state-of-the-art AI tools like OpenAI’s Whisper and open-source LLMs to transcribe, understand, and quiz users on content automatically.

> “Turn any video or document into an interactive quiz in seconds.”

---

## ✨ Features

- 🎥 **Video Input**: Paste a YouTube link or upload a local MP4.
- 📄 **Document Input**: Upload PDFs or text files.
- 🗣️ **Audio Transcription**: Uses OpenAI Whisper to transcribe video/audio.
- 🧠 **Quiz Generation**: Uses LLMs to create multiple-choice or true/false questions.
- 🌍 **Multilingual Support**: Supports content in multiple languages.
- 💾 **Quiz Download**: Export quizzes as PDF or text.
- ⚙️ **Video Caching**: Save time by avoiding repeated processing of the same videos.
- 📈 **Long Video Handling**: Smart chunking for efficient processing.
